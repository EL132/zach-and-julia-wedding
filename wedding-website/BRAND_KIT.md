# Brand Kit: Julia & Zach Wedding Website

## Brand Direction

The visual direction for the wedding website should be:

**European Garden Estate**

The site should feel elegant, romantic, botanical, and refined. The overall mood should be inspired by European villas, château gardens, formal outdoor dinners, soft white florals, manicured greenery, antique stationery, and warm ivory paper tones.

This should not feel like a generic modern wedding template. It should feel more like a quiet luxury estate wedding: old-world, garden-focused, polished, and intimate.

## Core Vibe

Use these words as the guiding brand language:

- European
- Garden
- Estate
- Botanical
- Romantic
- Refined
- Soft
- Formal
- Warm
- Timeless
- Editorial
- Old-world, but fresh

The site should feel formal enough for a wedding, but not cold or corporate. It should be romantic, but not overly cute. It should be beautiful and polished, but still personal and approachable.

## What the Brand Should Feel Like

The website should feel like:

> A refined European garden wedding website with warm ivory paper tones, layered sage and moss greens, classical serif typography, understated botanical details, and spacious editorial layouts.

The overall impression should be soft, elegant, and timeless.

## What to Avoid

Avoid design choices that feel:

- Too trendy
- Too playful
- Too techy
- Too bright
- Too pink
- Too modern-minimalist
- Too luxury-hotel / Gatsby
- Too rustic
- Too boho
- Too generic wedding-template

The brand should not rely heavily on script fonts, bright white backgrounds, harsh black text, neon greens, heavy shadows, or overly decorative wedding icons.

## Color Palette

The palette should be centered on warm neutrals, layered greens, and very restrained romantic accents.

### Primary Colors

| Name | Hex | Intended Use |
|---|---:|---|
| Warm Ivory | `#F8F4EA` | Main page background |
| Soft Cream | `#EFE7D8` | Section backgrounds, cards, subtle contrast areas |
| Garden Sage | `#A8B58A` | Soft accent color, dividers, secondary details |

## Color Distribution

The color distribution should be restrained and mostly neutral.

Recommended distribution:

- **70% ivory / cream**
- **20% greens**
- **8% dark text**
- **2% blush / antique gold accents**

The website should be primarily ivory, cream, sage, olive, and moss.

Blush and gold should be used sparingly. They should act as quiet supporting accents, not primary brand colors. Too much blush will make the site feel like a generic wedding website. Too much gold will make it feel overly glamorous or Gatsby-inspired.

## Color Usage Guidance

### Backgrounds

Use warm ivory as the main background color. Use soft cream for sections, cards, and areas that need gentle contrast.

The site should avoid pure white as the dominant background. The warmer ivory/cream tones are important for the antique paper, garden estate feeling.

### Text

Use deep moss or espresso for text rather than pure black.

Headings can use deep moss. Body text should generally use espresso for readability.

### Buttons and Links

Primary buttons should use deep moss or olive green with ivory text.

Secondary buttons can use transparent or cream backgrounds with moss borders and moss text.

### Accents

Use sage for soft borders, dividers, hover states, small decorative elements, and subtle section details.

Use blush and antique gold only for small details, such as a thin rule, monogram accent, tiny icon, or hover detail.

## Typography Direction

The typography should be serif-forward, elegant, and editorial.

The strongest recommendation is:

- **Display / Headings:** Cormorant Garamond
- **Body / UI:** Inter
- **Optional Accent:** Parisienne or no script font at all

## Typography Personality

The heading font should feel classical, romantic, and old-world. It should carry the wedding elegance.

The body font should be clean, readable, and modern enough to keep the website usable. It should not compete with the display font.

The optional accent/script font should be used extremely sparingly, if used at all. The brand should not depend on script typography. The elegance should primarily come from serif typography, spacing, color, and imagery.

## Typography Usage Guidance

### Headings

Use large, elegant serif headings with generous spacing.

Headings should feel editorial and refined, not overly bold or decorative.

### Hero Names

The couple’s names can be very large and serif-forward. This should likely be one of the strongest visual moments on the site.

### Body Text

Body text should be simple, readable, and calm.

Use a clean sans-serif font for paragraphs, labels, navigation, RSVP forms, travel information, and schedule details.

### Labels and Navigation

Use small uppercase sans-serif text with subtle letter spacing for navigation, section labels, and buttons.

This will help the site feel formal and polished without feeling cluttered.

## Overall Design Principles

The site should prioritize:

- Spacious layouts
- Warm neutral backgrounds
- Large serif typography
- Soft green accents
- Thin dividers and borders
- Editorial spacing
- Understated botanical details
- Elegant simplicity

The site should feel carefully designed, but not overdesigned.

When in doubt, choose restraint.
